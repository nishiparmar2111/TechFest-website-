TechFest 2026 Website

📌 Project Overview
The TechFest 2026 Website is a front-end web development project created as part of my college coursework.
It provides information about technical events, showcases past highlights through a gallery, and offers a simple user-friendly interface.
This project demonstrates the use of **HTML, CSS, and JavaScript** to build a responsive and interactive website.

 🚀 Features
* 🏠 Home page with navigation menu
* 📖 About page with event details
* 📅 Events page showcasing activities
* 🖼️ Gallery section with event highlights
* 📩 Contact page for user interaction
* 🎥 Background video for modern UI design
* 📱 Responsive layout for different devices

🛠️ Technologies Used
* **HTML5**– Structure of web pages
* **CSS3** – Styling, layout, and design
* **JavaScript** – Interactivity and dynamic effects
